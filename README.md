# 🚆 KMRL Intelligent Fleet Optimization System

**An AI-powered solution for Kochi Metro's complex train induction planning challenge**

---

## 🎯 Problem Statement

KMRL must decide nightly which of its 25 four-car trainsets will:
- Enter service 🚆
- Remain on standby ⏸️
- Go for maintenance 🔧

This involves 6 interdependent variables:
- Fitness Certificates  
- Job-Card Status  
- Branding Priorities  
- Mileage Balancing  
- Cleaning & Detailing  
- Stabling Geometry  

The current system is manual, error-prone, and not scalable.

---

## 🚀 Solution

An **AI-powered optimization system** using:
- Random Forest models
- LSTM neural networks
- Constraint-based optimization
- What-if scenario simulation

---

## 🧠 Features

- ✅ Intelligent train selection (Service / Maintenance / Standby)
- ✅ Predictive maintenance using ML
- ✅ Real-time optimization engine
- ✅ Scenario simulation (What-if analysis)
- ✅ Explainable AI outputs
